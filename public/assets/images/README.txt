Place your game images in this directory.

For the starter template, you'll need the following images:
- loading-background.png: A background for the loading screen
- loading-bar.png: A progress bar for the loading screen
- background.png: A background for the game
- logo.png: Your game logo
- button.png: A button image for UI
- player.png: A player character sprite
- platform.png: A platform for the player to stand on
- star.png: A collectible item
- bomb.png: An enemy or obstacle

You can create these images yourself or use placeholder images during development.
